gmu-senior_show-14
==================

## Description
Static site for the GMU 2014 Senior show.

## Turning on Sass.
1) navigate to the gmu-senior_show-14 folder in terminal
2) once in type sass --watch scss/main.scss:css/style.css
3) hit enter
4) it should work.
